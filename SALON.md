# AI 甩锅分析器

---

## 一句话介绍

**粘贴一段职场聊天记录，AI 帮你找出谁该背锅、整理证据链、生成优雅反击话术。**

---

## 这个产品解决什么问题？

职场中有一种痛，叫「明明不是我的锅，但说不清楚」：

- 复盘会上，对方一句「接口我周三就给了」，锅就飞到你头上
- 聊天记录几百条，翻半天找不到关键那几句
- 时间线混乱，说不清到底是谁 block 了谁
- 想反驳又怕显得不专业，不反驳又憋屈
- 没有现成的话术模板，每次都要临场发挥

**核心矛盾：信息不对称 + 表达能力不对等 = 老实人永远背锅。**

---

## 产品怎么用？

### 三步完成「反甩锅」

```
第一步：粘贴聊天记录（或上传截图，AI 自动识别文字）
         ↓
第二步：选择身份 + 点击「AI 深度分析」
         ↓
第三步：获得完整结果 → 继续追问 → 导出 PDF 留档
```

### 输出什么？

| 输出内容 | 说明 |
|----------|------|
| 锅归属判定 | 各方责任占比可视化（饼图 + 条形图） |
| 关键消息 | 最致命的 2-3 条证据，高亮标红 |
| 证据链时间线 | 按时间排列，标注延误/推诿/甩锅类型 |
| 反击话术 | 三种风格可选：高情商 / 阴阳怪气 / 对事不对人 |
| 改进建议 | 针对性建议，防患于未然 |

## 产品亮点

### 1. 双模式分析

- **本地快速分析**：关键词匹配，瞬间出结果，零流量消耗
- **AI 深度分析**：大模型语义理解，能读懂潜台词和隐含推诿

### 2. 不止分析，还能追问

分析完不够？继续追问 AI：
- 「如果领导追问我怎么回答？」
- 「帮我写一段更委婉的版本」
- 「对方可能怎么反驳？我怎么应对？」

AI 会基于上下文给出连贯回答。

### 3. 社区话术库

不知道怎么回？看看别人怎么说：
- 20+ 预置经典话术，覆盖转移话题、反击回怼、澄清甩锅、升级处理四大场景
- 用户可以贡献自己的经典回复，点赞收藏
- 遇到类似场景，直接复制使用

### 4. 数据看板

量化你的「反甩锅」成长：
- 总分析次数、本月使用量、连续使用天数
- 30 天使用趋势图
- 问题分类分布（什么类型的锅最常见）

### 5. 输入零门槛

- 支持粘贴钉钉/飞书/微信/邮件/会议纪要等任意格式
- 支持截图上传，AI 自动 OCR 识别文字
- 7 个内置场景示例，开箱即用

### 6. 结果可留档

- 分析历史自动保存，随时翻阅
- 一键导出为 PDF 或 PNG 图片
- 复盘时拿出来，有理有据

## 产品愿景

> **「不是教你甩锅，是帮你不背锅。」**

我们想做的不是挑起职场矛盾，而是：
- 帮助信息弱势方还原事实
- 让职场沟通更有据可依
- 用 AI 赋能每个不善言辞的老实人
- 最终推动更健康、更透明的职场协作文化
